# 🧲 Magnetic Hoppers

A highly optimized Vanilla+ Minecraft Data Pack that upgrades every hopper in your world. Hoppers will now actively scan a massive **7x7 area** (up to 3 blocks away horizontally and vertically) and magically "suck" dropped items directly into their inventory.

## Features
- **Save Iron:** 1 Hopper now does the work of 49 Hoppers!
- **Reduce Lag:** Fewer hoppers means significantly less lag on your world or server.
- **Perfect for Farms:** Automatically vacuums up drops from mob grinders, iron farms, and auto-harvesting setups.
- **Multiplayer Friendly:** Highly optimized to run seamlessly in the background without tick delay.

## Installation
Simply drop the `.zip` file into your world's `datapacks` folder and type `/reload`!
